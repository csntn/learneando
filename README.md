Proyectos de curiosidades y/o aprendizajes. 

### WorldCup2026
## Descripción

**Predicciones Mundial 2026** es una aplicación web que consulta los partidos de la Copa del Mundo y genera pronósticos automáticos utilizando un sistema de valoración de selecciones inspirado en el método Elo. La aplicación obtiene los encuentros disponibles mediante la API de API-Football y calcula probabilidades de victoria, resultados esperados y marcadores estimados a partir de una tabla interna de ratings de fuerza configurable.

El proyecto está desarrollado con **Node.js** y **Express** en el backend, mientras que el frontend utiliza **HTML, CSS y JavaScript vanilla**, ofreciendo una interfaz moderna, responsive y fácil de usar. Su objetivo es demostrar la integración de APIs deportivas, el desarrollo full-stack y la implementación de algoritmos básicos de predicción aplicados al fútbol internacional.

Las predicciones generadas tienen un carácter educativo y experimental, por lo que no deben considerarse pronósticos profesionales ni utilizarse como base para apuestas deportivas.
